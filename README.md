## About

I design systems where data serves decisions, not just dashboards. My work bridges rigorous software engineering with analytical depth — building reliable pipelines and decision-support architectures built to scale.

The goal is always the same: extract signal from noise, model outcomes before they happen, and make good decisions repeatable.
